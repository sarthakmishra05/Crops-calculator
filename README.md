This Python program, the **Crop Price Calculator**, is a utility designed to streamline the process of estimating the **selling price** of a specific harvested crop. 

---

### Key Functionality

The program operates by taking two primary inputs from the user to perform its calculation:

1.  **Crop Type and Unit Price:** The program requires the user to input the **current market price** for the chosen crop per standard unit of weight (e.g., price per kilogram, pound, or bushel). This value is crucial for the calculation's accuracy.
2.  **Weight of the Crop:** The user must enter the **total weight** of the crop they wish to sell, ensuring the weight unit is consistent with the unit price provided (e.g., if the price is per kilogram, the weight must also be in kilograms).

### Calculation Logic

The program applies a straightforward formula to determine the final selling price:

$$\text{Total Selling Price} = \text{Crop Unit Price} \times \text{Total Weight}$$

This calculation is executed instantly, providing the user with the expected revenue.
